# 🎓 Computer Recomendator
**La guía definitiva para tu primera gran compra tecnológica.**

Este proyecto nace de una problemática común entre estudiantes: el agobio por las especificaciones técnicas (RAM, CPU, GPUs) al elegir una computadora.  simplifica esta decisión a través de una experiencia visualmente atractiva y lenguaje humano.

### ✨ Características Principales
- **Quiz Inteligente:** 3-4 preguntas clave para definir tu perfil (Diseño, Programación, Estudio).
- **Traductor de Specs:** De "16GB LPDDR5X" a "Ideal para multitarea pesada y diseño".
- **Estética Minimalista:** Inspirado en el diseño limpio de Apple para una navegación intuitiva.
- **Comparador Visual:** Evalúa tus opciones favoritas lado a lado.
- **Curaduría de Influencer:** Recomendaciones basadas en experiencia real, no solo en números.

### 🛠️ Stack Tecnológico
- **Frontend:** Next.js + React
- **Styling:** Tailwind CSS + Framer Motion
- **Mobile First:** Optimizado para ser consultado desde redes sociales.
