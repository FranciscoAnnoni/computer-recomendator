       - `--muted`: convert #F5F5F7 to HSL (240 20% 96%)
       - `--muted-foreground`: convert #6E6E73 to HSL (240 2% 44%)
       - `--primary`: convert #0071E3 to HSL (211 100% 45%)
       - `--primary-foreground`: 0 0% 100% (white)
       - `--border`: convert #D2D2D7 to HSL (240 6% 83%)
       - `--accent`: same as primary (#0071E3)
       - `--accent-foreground`: 0 0% 100%
       Keep all other shadcn custom properties (card, popover, destructive, ring, etc.) with sensible values.

    2. Add `.dark` section in globals.css with inverted values:
       - `--background`: dark bg (~#000000 or #1D1D1F)
       - `--foreground`: light text (#F5F5F7)
       - `--muted`: dark surface
       - `--primary`: same blue accent
       - `--border`: dark border

    3. Update `tailwind.config.ts`:
       - Set `darkMode: "class"`
       - Extend `fontFamily` with `sans: ["var(--font-inter)", ...defaultTheme.fontFamily.sans]`

         - `"body"`:    ["1.0625rem", { lineHeight: "1.5", fontWeight: "400" }] (17px)
         - `"small"`:   ["0.75rem", { lineHeight: "1.4", fontWeight: "400" }]   (12px)

    4. Update `src/app/layout.tsx`:
       - Import Inter from `next/font/google` with `subsets: ["latin"]`, variable `--font-inter`
       - Wrap children with a ThemeProvider from `next-themes` (create inline or import):
         ```
         <ThemeProvider attribute="class" defaultTheme="system" enableSystem disableTransitionOnChange>
           {children}
         </ThemeProvider>
         ```
       - Apply `inter.variable` to the html element className
       - Set metadata: title "Computer Recomendator", description "Find the perfect laptop for your needs"

    5. Create `src/components/theme-provider.tsx`:
       ```typescript
       "use client"
       import { ThemeProvider as NextThemesProvider } from "next-themes"
       import { type ThemeProviderProps } from "next-themes"
       export function ThemeProvider({ children, ...props }: ThemeProviderProps) {
         return <NextThemesProvider {...props}>{children}</NextThemesProvider>
       }
       ```
