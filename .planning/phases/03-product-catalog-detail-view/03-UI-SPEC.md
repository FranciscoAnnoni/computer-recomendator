## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, tag pill gaps, inline padding |
| sm | 8px | Tag pill padding (px-2 py-0.5), card content internal gaps |
| md | 16px | Card content padding, filter chip row gaps, section internal padding |
| lg | 24px | Card-to-card gap in catalog list, overlay section gaps |
| xl | 32px | Page horizontal padding (mobile 16px, tablet 32px) |
| 2xl | 48px | Section breaks between quiz profile section and global catalog |
| 3xl | 64px | Page top/bottom padding |

Exceptions:
- Touch targets for "Ver más", "Filtrar", "Comprar Ahora" buttons: minimum 44px height (accessibility, WCAG 2.1 AA).
- Close button in detail overlay: 44px × 44px tap target.
- "Filtrar" button top bar: 40px height on mobile.

## Interaction States

### CatalogCard
- Default: shadow-sm
- Hover (desktop): shadow-md, slight scale(1.01) — 150ms ease
- Press/Active: scale(0.99)
- Focus-visible: ring-2 ring-primary ring-offset-2

### "Ver más" button
- Default: ghost, border accent, text accent
- Hover: bg-accent/10
- Loading: disabled + spinner icon (Lucide Loader2, animate-spin)

### "Comprar Ahora" button
- Default: bg-primary text-primary-foreground
- Hover: bg-primary/90
- Always opens new tab (rel="noopener noreferrer")

### Search input
- Focus: ring-2 ring-ring
- Typing: instant filter (debounce 200ms)

### Filter chips (ActiveFilterBar)
- Each chip: bg-secondary text-secondary-foreground border rounded-full, Lucide X icon right
- Remove on X tap: chip dismissed, filter cleared, catalog re-queries

### DetailOverlay enter/exit
- Enter: Framer Motion `y: "100%" → 0`, duration 300ms, ease: [0.32, 0.72, 0, 1] (Apple spring)
- Exit: `y: 0 → "100%"`, duration 250ms, ease: easeIn
- Backdrop: none (overlay is full-screen, no semi-transparent backdrop needed)

### Skeleton loading
- CatalogSkeleton: matches CatalogCard dimensions (image square + 4 content rows)
- Shimmer: bg-muted animate-pulse (tw-animate-css, already installed)
- Show 6 skeletons on initial load; replace with real cards on data resolve

## Accessibility Contract

- All interactive elements: minimum 44px touch target (WCAG 2.5.5).
- Images: meaningful alt text — use laptop name as alt value; placeholder div uses brand name.
- Color contrast: text on cards uses `text-foreground` (oklch dark on white) — confirmed WCAG AA.
- Muted text on cards (`text-muted-foreground`): used only for supplementary info (price, description), not for primary identification.
- Filter drawer: focus trap when open (shadcn Sheet handles this via Radix Dialog primitives).
- DetailOverlay: focus trap when open; Escape key closes.
- Star badge: aria-label="Recomendado por experto" on the badge element.
- "Comprar Ahora": rel="noopener noreferrer" + aria-label="Comprar {laptop name} (abre en nueva pestaña)".
- Search input: associated `<label>` visually hidden ("Buscar laptops").
- DetailOverlay close button: icon-only (Lucide X), no visible label. Must carry aria-label="Cerrar" as the sole text affordance. 44px × 44px tap target satisfies WCAG 2.5.5.

## Animation Contract

Source: Framer Motion (already installed). Apple-minimalist micro-interactions only.

| Interaction | Animation | Duration | Easing |
|-------------|-----------|----------|--------|
| DetailOverlay open | slide-up from y:100% to y:0 | 300ms | [0.32, 0.72, 0, 1] |
| DetailOverlay close | slide-down from y:0 to y:100% | 250ms | easeIn |
| FilterDrawer open | shadcn Sheet default (slide from bottom/right) | 300ms | default |
| CatalogCard hover scale | scale 1.0 → 1.01 | 150ms | ease |
| Skeleton shimmer | animate-pulse (tw-animate-css) | continuous | — |
| Catalog list mount | stagger fade-in, each card opacity 0→1, y 8→0 | 200ms per card, 40ms stagger | easeOut |
| Filter chip dismiss | opacity 1→0, width collapse | 150ms | easeOut |

No page-level transitions — overlay is additive, not a navigation. Router stays on /catalog.
