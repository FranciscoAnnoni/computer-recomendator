Create 9 SVG illustration React components. Each component is a `.tsx` file that exports a default function returning an `<svg>` element.

**Shared SVG properties for ALL illustrations:**
- viewBox="0 0 120 120"
- width and height as props with defaults: `width={120} height={120}`
- Monochromatic: use `stroke="currentColor"` (inherits from parent text color)
- strokeWidth="1.5", fill="none", strokeLinecap="round", strokeLinejoin="round"
- className prop forwarded to the root `<svg>` element
- Style: flat/isometric line art, minimalist geometric shapes

**Each illustration file follows this pattern:**
```typescript
export default function ProductivityIllustration({ width = 120, height = 120, className }: { width?: number; height?: number; className?: string }) {
  return (
    <svg viewBox="0 0 120 120" width={width} height={height} fill="none" stroke="currentColor" strokeWidth="1.5" strokeLinecap="round" strokeLinejoin="round" className={className} aria-hidden="true">
      {/* SVG paths here */}
    </svg>
  );
}
```
