**A. Create `src/components/quiz/quiz-result.tsx`:**
- "use client" directive
- Props: `selections: [string | null, string | null, string | null]`, `onRetry: () => void`
- Internal state: `profile` (ProfileResult | null), `laptops` (Laptop[]), `loading` (boolean, starts true), `error` (string | null)
- On mount (useEffect): call `fetchProfile(selections[0] as Workload, selections[1] as Lifestyle, selections[2] as Budget)`, then `fetchLaptopsByIds(profile.laptop_ids)`. Store results. Set loading=false. On error: set error message, log the (workload, lifestyle, budget) combination to console.error for debugging. Also write completed profile + laptops to `PROFILE_STORAGE_KEY` in localStorage for Navbar avatar consumption.
- **Loading state:** Render `<ResultSkeleton />`
- **Error state:** `<div role="alert" className="text-center py-12">` containing:
  - Heading: `<h2 className="text-subhead font-medium text-foreground">No encontramos tu perfil</h2>`
  - Body: `<p className="text-body text-muted-foreground mt-2">Algo salio mal al buscar tu perfil. Intenta hacer el quiz de nuevo.</p>`
  - CTA: `<Button onClick={onRetry} className="mt-4">Volver a intentarlo</Button>`
- **Empty state (profile found but no laptops):** `<p className="text-body text-muted-foreground text-center py-12">Aun no hay laptops para este perfil. Vuelve pronto.</p>`
- **Success state:** Wrap in `<motion.div initial={{ opacity: 0 }} animate={{ opacity: 1 }} transition={{ duration: 0.4, ease: "easeOut" }}>`:
  - Profile header: `<h2 className="text-subhead font-medium text-foreground">{profile.profile_name}</h2>` + `<p className="text-body text-muted-foreground mt-1">{profile.profile_description}</p>`
  - Section heading: `<h3 className="text-body font-medium text-foreground mt-8">Tus 5 mejores opciones</h3>`
  - Laptop list: `<div className="flex flex-col gap-4 mt-4">` mapping `laptops` to `<ResultLaptopCard key={laptop.id} laptop={laptop} />`
- Export: `export function QuizResult({ selections, onRetry }: QuizResultProps)`
