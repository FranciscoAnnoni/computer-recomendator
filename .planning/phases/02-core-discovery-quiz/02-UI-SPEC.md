## Interaction Contract

### Carousel Mechanic

- Layout: center card large (280px mobile / 320px md+), left and right cards partially visible — peek 60px from each edge
- Navigation: ← → arrow buttons, 44px tap target, positioned vertically centered on the card row
- Center card = selected: `border-2 border-primary ring-2 ring-primary/20 scale-[1.03]` transition
- Non-center cards: `opacity-60 scale-[0.92]` — dimmed and shrunk
- Swipe gesture on mobile: drag threshold 40px → snap to adjacent card (Framer Motion `drag="x"` with `dragConstraints`)
- Snap behavior: CSS `scroll-snap-type: x mandatory` as fallback; Framer Motion JS snap as primary

### Step Transitions

- Direction: slide left on Next (x: 0 → -100%), slide right on Back (x: 0 → +100%)
- New step enters from opposite direction simultaneously
- Duration: 300ms
- Easing: `easeInOut` (Framer Motion default curve)
- Implementation: `AnimatePresence` with `mode="wait"`, `key={currentStep}`

### Result Transition

- Trigger: after Supabase profile+laptops fetch resolves
- Animation: fade in (`opacity: 0 → 1`), duration 400ms, easing `easeOut`
- Loading state: skeleton cards (3 animated pulse lines per card) while fetch is in-flight

### Profile Avatar (Navbar)

- Appears on right side of Navbar, replaces "Find My Laptop →" CTA after quiz completion
- Shape: 32px circle, `ring-2 ring-primary` border
- Tapping opens a Sheet (base-ui render prop pattern, matching existing `sheet.tsx` pattern)
- Sheet content: profile name (subhead 28px/500), profile description (body 17px/400), 3 selected-card thumbnails (48×48px), "Rehacer quiz" button (primary)

### Progress Indicator

- Position: directly below step question heading, above carousel
- Format: "Paso X/3" label (12px/400, muted-foreground) + segmented bar
- Segmented bar: 3 equal segments separated by 4px gaps, 4px height, 8px border-radius
- Active segments: `bg-primary`, inactive: `bg-border`
- Width: full container width

### Back Arrow

- Position: top-left of quiz page, 16px from container edge
- Icon: `ChevronLeft` (lucide, size-5)
- Tap target: 44px × 44px
- Hidden on Step 1 (no previous step)
- Preserves previous carousel selection on return

## Accessibility Contract

- All interactive elements: minimum 44×44px touch target (WCAG 2.5.5) — Source: RNF2.2
- Color contrast: foreground on background ≥ 4.5:1 (WCAG AA) — confirmed by Apple neutral palette
- Carousel arrows: `aria-label="Opción anterior"` and `aria-label="Opción siguiente"`
- Step progress: `role="progressbar"`, `aria-valuenow={step}`, `aria-valuemin="1"`, `aria-valuemax="3"`, `aria-label="Progreso del quiz"`
- Selected card: `aria-pressed="true"` on the card button
- Loading state: `aria-live="polite"` region with sr-only text "Buscando tus laptops recomendadas…"
- Error state: `role="alert"` on error container
- Keyboard: carousel navigable with ← → arrow keys when carousel has focus; Enter/Space to confirm selection
- Focus ring: Tailwind `focus-visible:ring-2 focus-visible:ring-primary` on all interactive elements
