**Component signature:**
```typescript
interface FeedbackDialogProps {
  open: boolean;
  onOpenChange: (open: boolean) => void;
}
export function FeedbackDialog({ open, onOpenChange }: FeedbackDialogProps)
```

**State:**
```typescript
const [message, setMessage] = useState("");
const [honeypot, setHoneypot] = useState("");
const [status, setStatus] = useState<"idle" | "submitting" | "success" | "error">("idle");
const [errorMsg, setErrorMsg] = useState("");
```

**handleSubmit logic (in order):**
1. `e.preventDefault()`
2. If `honeypot !== ""` → silently close: `onOpenChange(false)`, reset state, return
3. If `message.trim() === ""` → return (textarea has `required` so this is a safety check only)
4. `setStatus("submitting")`
5. `const { error } = await supabase.from("feedback").insert({ message: message.trim() })`
6. If `error` → `setStatus("error")`, `setErrorMsg("Algo salió mal, intenta de nuevo")`, return
7. On success:
   - Increment localStorage: `const count = parseInt(localStorage.getItem(FEEDBACK_COUNT_KEY) ?? "0", 10); localStorage.setItem(FEEDBACK_COUNT_KEY, String(count + 1));`
   - `setStatus("success")`
   - `setTimeout(() => { onOpenChange(false); setMessage(""); setStatus("idle"); }, 1500)`

**JSX structure:**
```tsx
<Dialog.Root open={open} onOpenChange={onOpenChange}>
  <Dialog.Portal>
    <Dialog.Backdrop
      className="fixed inset-0 z-[100] bg-black/60 transition-opacity duration-200 data-ending-style:opacity-0 data-starting-style:opacity-0"
    />
    <Dialog.Popup
      className="fixed z-[101] left-1/2 top-1/2 -translate-x-1/2 -translate-y-1/2 w-full max-w-md bg-background rounded-2xl shadow-2xl p-6 transition duration-200 ease-in-out data-ending-style:opacity-0 data-starting-style:opacity-0 data-starting-style:scale-95 data-ending-style:scale-95"
    >
      <Dialog.Title className="text-base font-semibold text-foreground mb-4">
        Deja tu comentario
      </Dialog.Title>

      {status === "success" ? (
        <p className="text-center text-foreground py-6">¡Gracias por tu comentario! ❤️</p>
      ) : (
        <form onSubmit={handleSubmit} className="flex flex-col gap-4">
          {/* Honeypot — visually hidden but NOT display:none so bots fill it */}
          <input
            type="text"
            name="website"
            value={honeypot}
            onChange={(e) => setHoneypot(e.target.value)}
            tabIndex={-1}
            autoComplete="off"
            aria-hidden="true"
            className="absolute opacity-0 pointer-events-none w-0 h-0 overflow-hidden"
          />

          <textarea
            value={message}
            onChange={(e) => setMessage(e.target.value)}
            placeholder="Cuéntanos qué piensas..."
            required
            rows={4}
            className="w-full rounded-lg border border-border bg-background px-3 py-2 text-sm text-foreground placeholder:text-muted-foreground resize-none focus:outline-none focus:ring-2 focus:ring-primary"
          />

          {status === "error" && (
            <p className="text-sm text-destructive">{errorMsg}</p>
          )}

          <div className="flex justify-end gap-2">
            <Dialog.Close
              render={
                <button
                  type="button"
                  className="px-4 py-2 text-sm text-muted-foreground hover:text-foreground transition-colors"
                />
              }
            >
              Cancelar
            </Dialog.Close>
            <button
              type="submit"
              disabled={status === "submitting"}
              className="px-4 py-2 text-sm font-medium bg-primary text-primary-foreground rounded-lg hover:opacity-90 transition-opacity disabled:opacity-50"
            >
              {status === "submitting" ? "Enviando..." : "Enviar"}
            </button>
          </div>
        </form>
      )}
    </Dialog.Popup>
  </Dialog.Portal>
</Dialog.Root>
```

**Do NOT** use `display:none` on the honeypot — use `opacity-0 pointer-events-none` so CSS hides it from humans but bots still see and fill it.
**Do NOT** import Dialog from sheet.tsx — import directly from `@base-ui/react/dialog`.
