## MercadoLibre Argentina — Programa de Afiliados
1. Go to `https://www.mercadolibre.com.ar/l/afiliados`
2. Sign in with active MercadoLibre account (must also have Mercado Pago)
3. Must be 18+ years old
4. Complete registration form with social media/content channel info
5. Receive confirmation email and activate
6. Commission: 2-4% on computers/electronics
7. Payment: credited to Mercado Pago 60 days after delivery

### Generating MercadoLibre Affiliate Links
- After approval, browse products on mercadolibre.com.ar while logged in
- A "blue bar" / affiliate toolbar appears on product pages
- Click to generate the affiliate-tagged URL
- Copy the full URL — this is what goes into `laptops.affiliate_link`
- Note: exact URL parameter format is not publicly documented; the dashboard generates opaque redirect URLs

## Amazon Associates (US Program)
1. Go to `https://affiliate-program.amazon.com/`
2. Sign in or create Amazon.com account
3. Complete Associate profile (website URL, content type, expected traffic)
4. Get Associate Store ID (format: `yourname-20`)
5. Begin generating links immediately via SiteStripe toolbar
6. Full approval after first 3 qualifying sales within 180 days
7. Commission: ~4-5% on electronics/laptops

### Amazon Affiliate Link Format
```
https://www.amazon.com/dp/{ASIN}/?tag={associate-id}
```
Example: `https://www.amazon.com/dp/B0BN9L5FKK/?tag=mysite-20`

### Important: No Amazon Argentina
- `amazon.com.ar` does NOT exist
- Argentine customers buy from `amazon.com` with international shipping
- Use the US Amazon Associates program, not a regional one

## Updating Laptop Records in Supabase
After generating affiliate links for each laptop:

```sql
-- Update a single laptop's affiliate link
UPDATE laptops SET affiliate_link = 'https://www.amazon.com/dp/B0BN9L5FKK/?tag=yoursite-20' WHERE name = 'Laptop Name Here';
```

Or use the Supabase Dashboard > Table Editor > laptops > click the row > edit `affiliate_link` field.

### Important: Do Not Leave affiliate_link Empty
- An empty `affiliate_link` (`''`) renders `<a href="">` which navigates to the same page
- For laptops entered before affiliate registration, use the plain product URL (no affiliate tag)
- Update to the tagged URL after registration is complete
